REST API Documentation

Programmatically detect cancer variants via JSON API.

POST /api/detect

Submit a single variant for cancer detection analysis.

Request Body (JSON)
{
  "chromosome": "17",
  "position": 43044295,
  "gene": "BRCA1",
  "variant_type": "Missense",
  "ref_allele": "G",
  "alt_allele": "A",
  "clinical_significance": "Pathogenic"
}

{

    "chromosome": "17",
    "position": 43044295,
    "gene": "BRCA1",
    "variant_type": "Missense",
    "ref_allele": "G",
    "alt_allele": "A",
    "clinical_significance": "Pathogenic"
  },
  "cancer_detection": {
    "is_pathogenic": true,
    "cancer_probability": 85.3,
    "risk_level": "HIGH"
  },
  "cancer_type_prediction": {
    "gene": "BRCA1",

    },
    "top_cancer_type": "Breast Cancer",
    "top_confidence": 81.7

}
Parameters
Field Type Default Description
chromosomestring"1"Chromosome (1-22, X, Y, MT)
positionint0Genomic position
genestring"UNKNOWN"Gene symbol (e.g. BRCA1, EGFR)
variant_typestring"SNP"Variant classification
ref_allelestring""Reference allele
alt_allelestring""Alternate allele
clinical_significancestring""ClinVar significance label
cURL Example
curl -X POST http://localhost:5000/api/detect \
  -H "Content-Type: application/json" \
  -d '{"chromosome":"17","position":43044295,"gene":"BRCA1","variant_type":"Missense","ref_allele":"G","alt_allele":"A"}'
